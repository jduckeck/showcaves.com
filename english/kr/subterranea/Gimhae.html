History


Description

김해와인동굴 (Gimhae Wine Cave) seemed at the first glimpse to be one of those wine shops which call theme self cave, probably because cave is the French term for a wine cellar. But that's completely wrong, this is a real underground structure, and it is weird.

It started all with a railroad tunnel called Saengrim Tunnel, which was abandoned. Now there was a great underground tunnel which was available for use. Raspberry wine is a local specialty of Gimhae. The wine is produced and then matured and store in the tunnel, a 215 m long section is actually used as wine cellar.

The rest of the tunnel is accessible for visitors. The LED light tunnel is 70 m long, then Berry’s Strawberry Village is 60 m long, and finally the Wine tasting and sales is in the next 90 m. The berry wine based little theme park uses the raspberry girl Berry for advertising, a cartoon character which expresses raspberries.
