Description

The name Cave Santorini Spa is weird, nevertheless, it actually explains everything. This is a spa which was built into historic caves which were used as cave houses as well as cellars. As this venue is intended for foreign tourists, it has an English name but no Greek name.

The spa is open to everyone, but as it is located inside the premises of the Dome Santorini Resort & Spa the visitors are primarily hotel guests. We have not listed the hotel, though, because it does not offer cave rooms.
