Caving Gear

    Caving gear is all kind of gear one needs for caving. This includes climbing/mountaneering gear, lights, and special caving gear.

    Many useful things developed for climbing are used by cavers. But often they had to be adapted to the different conditions. For example, cavers often go down into the cave first and climb up to the entrance at the end of the trip. This means, climbing up an existing rope - which is unthinkable for mountaineering - is very common in caving! Also the use of rope ladders is very common.

    A special problem is the light. As light is essential for a save caving trip, it is absolutely necessary to take several light sources with you. If possible, take different kinds of light with you, they may have different pros and cons. So if it is too windy for a carbide lamp, the electric light will still work. But if the tour takes very long, it would be hard to carry enough spare batteries.
