Description

淮北国家矿山公园 (Huaibei National Mine Park) is a former coal mine, which has been transformed into a park. Enormous mining machinery is still there, which forms an open air museum. The Mining Museum has interactive exhibits which explain the coal mining history and the techniques. There is also a replica mine, which reconstructs the actual underground working deceptively real.
