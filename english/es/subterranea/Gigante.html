Description

It seems bot versions of the name are used, Cuevas del Gigante and Cueva El Gigante, both meaning Giant Cave. Actually this cave is not big enough to be called that, but it is quite interestin. It starts with the uniquie location, in a mountain rdige above the village Temisas. Then it is a through-cave which was created artificially by the Guanche, and it was either used for living or to store things. The main passage has numerous chambers on both sides. They are not really gigantic, but more comfortable than a tent.
