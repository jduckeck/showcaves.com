Github Download der gesamten Website

Wenn Sie über die Website diskutieren oder bei der Website mithelfen möchten, wir verwenden Slack and Github für die Verwaltung der Website. Wenn Sie also eine Offline-Version wünschen, gehen Sie einfach zu Github und laden Sie die vollständige Website herunter. Öffnen Sie die Datei index.html von der Festplatte und genießen Sie! Wenn Sie mit uns zusammenarbeiten möchten, senden wir Ihnen gerne eine Einladung für Slack, damit wir alles, was damit zusammenhängt, besprechen können. Wenn Sie sich fit genug fühlen, html zu bearbeiten, können wir über Schreibzugriff auf den Github sprechen. Dies ist ziemlich gefährlich, daher werden wir dies nur tun, wenn Sie wirklich interessiert sind und nach einiger Zeit in Slack.


Natürlich können auch die Betreiber von Schauhöhlen oder Bergwerken ihr eigenes Objekt aufnehmen lassen. Diese Seiten und der Update sind kostenlos! Beachten Sie aber, dass dies keine kostenlosen Werbeseiten für Sie sind, die Entscheidung über den Inhalt liegt ausschließlich bei uns. Wir erstellen keine Werbeseiten, nehmen keine Gefälligkeitslinks in den Text auf und schalten auch keine Banner. Wir sind interessiert, uns möglichst auf harte Fakten zu beschränken. Wenn ihre Sehenswürdigkeit diesen Namen verdient können wir Sie vieleicht überraschen wieviel eine objektive Beschreibung bringt.
