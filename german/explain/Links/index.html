Wenn Sie diese Seiten bereits besucht haben, werden Sie Schwierigkeiten haben, die Linkseiten wiederzufinden. Diese Linkliste wurde aus verschiedenen Gründen entfernt. Links haben leider die Angewohnheit sehr schnell zu veralten, sodass die Wartung einer manuell erstellten Linkliste einen erheblichen Aufwand bedeutet. Weil diese Arbeit zudem recht langweilig und unangenehm ist, wird sie ungerne gemacht, und so veralten die Seiten bald bis zu einem Grad, in dem sie nicht mehr hilfreich sind. Unsere Besucher haben offensichtlich Probleme dies so hinzunehmen, sodass wir sehr verärgerte Reaktionen bekommen haben. Wir haben deshalb den Sinn dieser Seiten neu überdacht.

showcaves.com ist so aufgebaut, dass jede unterirdische Sehenswürdigkeit ihre eigene Seite hat. Dort werden jeweils die Links zu diesem Ort angegeben. Dies wird natürlich auch in Zukunft so bleiben. Übersichtsseiten für Regionen oder Länder besitzen ebenfalls Linklisten, die ebenfalls erhalten bleiben. In der Linkliste hier wurden zusätzliche Links eingestellt, die innerhalb der Struktur von showcaves.com keinen Platz hatten. Und das zeigt auch ihre wesentliche Eigenschaft: sie hatten nur einen sehr wagen Bezug zum Inhalt von showcaves.com.

Aus diesem Grund haben wir uns entschlossen diese Abteilung zu schließen, die Seiten wurden vollständig entfernt. Wenn Sie einen Link zu einem Thema suchen das nicht auf showcaves.com behandelt wird, benutzen Sie bitte eine der inzwischen reichlich vorhandenen Speleo Link Listen oder Google.
