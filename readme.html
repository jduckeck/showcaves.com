Welcome to the CD-ROM version of showcaves.com. Willkommen zur CD-ROM Version von showcaves.com.
The fact that you can read this page shows me, you know how to open the file. Still, this is originally a web site, and so you should know a few details how to use it. Die Tatsache, dass Sie diese Datei lesen können, zeigt mir, dass Sie es geschafft haben die CD-ROM zu öffnen. Da es sich dabei jedoch eigentlich um eine Website handelt, sollten Sie noch einige Details zur Benutzung kennen.
  • You can use any browser to view showcaves.com, although it may look better on newer browsers.
  • To open the CD-ROM, just insert it into the drive, and on all newer Windows™ systems it will load automatically and open your default browser.
    If not: select in the file menu of your browser the option to open a file from disk. Then look for index.html on your CD-ROM drive and open it.
  • You can bookmark pages (set favorites) as you would do on the website.
  • Navigation will work exactly like you know from the website. But some links will only work with an connection to the internet:
    • The search forms at the foot of the pages need a search engine, which is Google. So it will work only online. And unfortunately it will find only online resources so the results will also be online.
    • Links with the small Earth symbol are links into the internet and will only work with online connection.
    • Some links with the small book symbol are links into the internet and will only work with online connection.
    • The Submit Information opens a connection to the server to send the data you provided and will work only online. But you may fill in the form offline. The same applies of course to the Submit Link form.
  • Sie können jeden Browser zum Lesen von showcaves.com benutzen, obwohl die Darstellung auf neueren Browsern natürlich besser ist.
  • Um die CD-ROM zu öffen, legen Sie sie einfach in Ihr Laufwerk und auf allen neueren Windows™ Systemen wird diese Seite in Ihrem Standardbrowser geöffnet.
    Wenn nicht: wählen Sie im Datei-Menü Ihres Browsers die Option eine lokale Datei zu öffnen. Öffnen Sie die Datei index.html auf Ihrer CD-ROM.
  • Sie können bookmarks (Favoriten) setzen, genau so wie Sie das von der Website gewöhnt sind.
  • Die Navigation funktioniert wie auf der Website. Aber einige Links funktionieren nur mit einer Verbindung zum Internet:
    • Das Suchformular am Fuß jeder Seite funktioniert unter Verwendung der Suchmaschine Google, also nur in Verbindung mit diesem Dienst. Außerdem können bei der Suchen nur die Seiten der Website gefunden werden.
    • Links mit dem kleinen Weltkugelsymbol sind Links ins Internet und funktionieren nur online.
    • Einige Links mit dem kleinen Buchsymbol sind Links ins Internet und funktionieren nur online.
    • Das Online Formular speichert die Eingaben auf dem Server, ohne Verbindung ist das nicht möglich. Allerdings kann das Formular offline ausgefüllt werden. Das gleiche gilt für das Link Formular.

  • Diese CD-ROM ist ausschließlich für Iher persönliche Benutzung. Sie könne so viele Kopien, Backups, Installationen davon machen wie Sie möchten, allerdings ausschließlich für Ihre persönliche Benutzung.
